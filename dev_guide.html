Developer guidelines

This homepage is compiled using jemdoc, but the mod which allows using MathJax for typesetting math. If you want to compile html:

  1. check-out jemdoc+MathJax repo, and update your PYTHONPATH to include the this repo,

  2. check-out acado.github.io repo, cd to the appropriate folder and

  3. type make. Voila!
