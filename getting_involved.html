Getting involved

In case you find a bug, want to contribute or simply have an idea about an enhancement, you can make a ticket on the issue tracker. The source code of the toolkit is hosted on GitHub. In case you are willing to contribute and/or solve a bug:

  • make your own fork on GitHub,

  • work on the code and

  • make a pull request on GitHub.
